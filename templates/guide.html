📖 Overview

Gifomatic is a powerful web application that automatically converts videos into GIFs using intelligent scene detection. Simply upload a video, and Gifomatic will:

  1. Analyze your video to detect scene changes
  2. Split the video into clips at scene boundaries
  3. Convert each clip into an optimized GIF
  4. Stream results to you in real-time
💡 Key Benefit: No manual work required! Gifomatic intelligently detects where scenes change in your video, ensuring each GIF captures a complete moment.

🚀 Getting Started

Before using Gifomatic, make sure you have a video file ready. Check the supported formats below:

MP4 AVI MOV MKV WebM FLV WMV

Video Length Any length (longer videos take more time)
Resolution Any (output scaled based on your settings, 320-1920px)

📤 Step 1: Upload Video

1

Click the Upload Area

On the home page, click the "Choose a video file" button or drag and drop your video file onto the upload area.

2

Select Your Video

Browse your computer and select a video file. The filename will appear below the upload area.

3

Click Process Video

Once a file is selected, the "Process Video" button becomes active. Click it to start processing.

💡 Tip: If you've processed this video before with the same settings, Gifomatic will automatically load it from cache - no reprocessing needed!

⚙ Step 2: Configure Settings

Before processing, you can customize how your GIFs are generated. Click on "GIF Settings" to expand the configuration panel.

Available Settings

Max Duration 1-30 seconds. Longer scenes are split into chunks of this duration. Default: 5s
GIF Width 320px to 1920px. Higher = better quality, larger files. Default: 480px
Frame Rate (FPS) 5-30 FPS. Higher = smoother, larger files. Default: 10 FPS
Scene Sensitivity 10-60. Lower = more scenes detected. Default: 30

Quick Presets

Use presets for common use cases:

Small Files

320px, 5 FPS, 3s max. Best for quick sharing, minimal file size.

Balanced

480px, 10 FPS, 5s max. Good quality with reasonable file size. (Default)

HD Quality

720px, 15 FPS, 5s max. High quality for presentations.

Maximum

1080px, 24 FPS, 10s max. Best quality, largest files.

⚠ File Size Warning: Higher quality settings create significantly larger GIFs. A 1080px, 24 FPS GIF can be 10-50x larger than a 320px, 5 FPS version.

🖼 Step 4: Browse GIFs

Once processing begins, GIFs appear in a responsive grid layout. The interface has two tabs:

🎨 Original GIFs

All GIFs generated from your video, displayed in chronological order. This is where you select GIFs for merging or downloading.

🔗 Merged GIFs

GIFs you've created by merging multiple original GIFs. These are saved and persist even after refresh.

GIF Card Information

Each GIF card shows:

  • GIF Preview: Animated preview of the GIF
  • Filename: Shows at the bottom (e.g., clip_0001.gif)
  • Download Icon: Appears on hover (top-right corner)
  • Selection State: Purple border when selected

✅ Step 5: Select GIFs

Selection Methods

🖱 Click to Select

Click anywhere on a GIF card (except the image itself) to toggle selection. Selected GIFs show a purple border and checkmark overlay.

🗂 Select All

Click the "Select All" button in the action bar to select all GIFs at once.

❌ Unselect All

Click "Unselect All" to clear all selections.

Action Bar

The floating action bar at the bottom shows:

  • Selection Count: "X selected" or total GIF count
  • Select All / Unselect All: Batch selection buttons
  • Download Selected: Download only selected GIFs
  • Download All: Download all GIFs in current tab
  • Merge Selected: Combine selected GIFs (requires 2+)

🔗 Step 6: Merge GIFs

The merge feature combines multiple GIFs into one continuous animation, playing them in sequence.

How to Merge

1

Select GIFs

Select 2 or more GIFs you want to merge. They will be combined in filename order (clip_0001, clip_0002, etc.).

2

Click Merge Selected

The button activates when 2+ GIFs are selected. Click it to start merging.

3

View Result

After merging, you're automatically switched to the "Merged GIFs" tab where your new GIF appears.

💡 Tip: Selections are automatically cleared after merging, so you can quickly select and merge another set.

🗃 Previous Jobs

Gifomatic saves all processed videos. You can reload them without reprocessing.

How It Works

  • Automatic Caching: When you upload a video, Gifomatic creates a unique fingerprint including your settings. If you upload the same video with the same settings, it loads from cache instantly.
  • Settings-Aware: Different settings create different cache entries. Processing the same video at 480px vs 1080px creates two separate cached results.
  • Previous Jobs Section: Shows all previously processed videos below the upload area.
  • One-Click Load: Click any job to load its GIFs immediately.

What's Saved

  • All original GIFs (clip_xxxx.gif)
  • All merged GIFs (merged_xxxx.gif)
  • Video fingerprint + settings combination for cache matching
💡 Tip: Processing the same video with different settings will create a new job. To reprocess with identical settings, delete the job folder from output/ and its entry from cache.json.

💡 Tips & Best Practices

🎬 Video Quality

Higher quality videos produce better GIFs. If possible, use the original video file rather than a compressed version.

⏲ Video Length

For quick results, use shorter videos (under 5 minutes). Longer videos work but take more time to process.

🎨 Scene-Rich Videos

Videos with lots of cuts (like trailers or compilations) work great! Each scene becomes its own GIF.

🔗 Strategic Merging

Merge GIFs that tell a story together. Select them in the order you want them to play.

💾 File Size

GIF file sizes vary based on content. High-motion scenes create larger files.

💻 Browser Performance

If you have many GIFs, older devices may lag. Use the gallery view for smoother playback.

🔧 Troubleshooting

❓ Video won't upload

Solution: Check that your file format is supported (MP4, AVI, MOV, MKV, WebM, FLV, WMV). Try a different browser if issues persist.

❓ Processing seems stuck

Solution: Check the terminal/console for error messages. Scene detection can take time for long videos. FFmpeg must be installed and accessible.

❓ GIFs not appearing

Solution: Ensure FFmpeg is properly installed. Check terminal for "FFmpeg not found" errors. Restart the application after installing FFmpeg.

❓ Poor GIF quality

Solution: Use the "GIF Settings" panel to increase width (up to 1920px) and FPS (up to 30) before processing. Note that higher quality means larger file sizes.

❓ Cache not working

Solution: Ensure cache.json is writable. The video fingerprint is based on the first 10MB - identical starts but different content won't match.

❓ Merge fails

Solution: Ensure all selected GIFs exist and are valid. Check terminal for FFmpeg error messages.

❔ Frequently Asked Questions

Q: Is there a limit to how many GIFs I can create?

A: No limit! The number of GIFs depends on your video length and scene changes.

Q: Can I change the GIF quality settings?

A: Yes! Use the "GIF Settings" panel before processing to adjust duration (1-30s), width (320-1920px), FPS (5-30), and scene sensitivity.

Q: Why are some GIFs shorter than my max duration?

A: GIFs are split at scene boundaries. If a scene is only 2 seconds, the GIF will be 2 seconds regardless of the max duration setting.

Q: Can I process multiple videos at once?

A: Currently, one video at a time. Wait for processing to complete before uploading another.

Q: Where are my GIFs stored?

A: In the output/ folder, organized by job ID. Each job has its own subfolder.

Q: Can I run this on a server?

A: Yes! Use Docker or set FLASK_ENV=production. See the README for deployment instructions.

Q: Is my video data private?

A: Yes! All processing happens locally on your machine. Videos are deleted after processing; only GIFs are kept.
